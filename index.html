What is ankylo

KEYS SECURED.
WORKFLOW INTACT.

ankylo is a command-line API key vault built for developers who want their secrets stored securely without leaving the terminal. Your entire vault lives in a single encrypted file on disk — no cloud sync, no accounts, no browser required.

Every entry is protected by a master password you choose at initialization. That password never touches disk — instead, it's run through Argon2id to derive an encryption key, which then encrypts your vault with AES-256-GCM. Even if someone gets your vault file, they get nothing useful.

Need to grab a key fast? One command copies it straight to your clipboard. Need to bulk-import from a dotenv file? ankylo reads it. Need to export back out? ankylo writes it. Everything stays in the terminal, everything stays encrypted at rest.

Security architecture

BUILT ON
SERIOUS CRYPTO.

🔑
KDF
Argon2id
Memory-hard password hashing resistant to GPU and side-channel attacks. time_cost=3, memory_cost=64MB, parallelism=2. Your password never hits disk.
🔒
Encryption
AES-256-GCM
Authenticated encryption with associated data. Every vault save generates a fresh nonce, providing both confidentiality and tamper detection.
💾
Storage
Atomic Writes
Vault updates write to a temp file first, then atomically replace the old one. No partial writes, no corruption risk on crash or power failure.
📋
Access
Clipboard Delivery
Retrieved keys are copied directly to your clipboard — never printed to terminal output, protecting against shell history and screen captures.
📁
File I/O
Dotenv Compatible
Import entries from NAME=KEY formatted files in bulk. Export your vault back out to the same format for use in other tools.
⚡
Interface
Typer CLI
Built with Typer for a fast, clean command-line experience with auto-generated help text and argument validation out of the box.

Command reference

EVERYTHING
YOU NEED.

ankylo init
—
Create and initialize a new encrypted vault. Sets your master password. Run this once before anything else.
ankylo add
<name>
Add a new API key entry to the vault. Prompts securely for the key value. Optionally replaces an existing entry by name.
ankylo get
<name> [--show]
Retrieve a key by name. Copies it to clipboard. Use --show to display the key value directly in terminal.
ankylo get-index
<index> [--show]
Retrieve a key by its numeric index. Useful for quick access when you know the list order.
ankylo list
[--show] [--output <file>]
List all vault entries. Add --show to reveal keys. Use --output to write entries to a file.
ankylo add-file
<path>
Bulk-import entries from a NAME=KEY formatted file. Processes every line and adds each as a vault entry.
ankylo delete
<name>
Delete a vault entry by name. Requires confirmation before removal. This action is permanent and cannot be undone.
ankylo delete-index
<index>
Delete a vault entry by its index. Requires confirmation. Permanent operation — use with care.
ankylo delete-vault
—
Permanently destroy the entire vault and all stored keys. Requires master password verification before deletion.

Installation

UP IN
SECONDS.

Option 1 — install from GitHub
$ pip install git+https://github.com/Iced-code/ankylo.git
Option 2 — clone and install
$ git clone https://github.com/Iced-code/ankylo.git
$ cd ankylo
$ pip install .
01
ankylo init
Create your vault and set a master password
02
ankylo add openai
Add your first entry — you'll be prompted for the key value
03
ankylo get openai
Key copied to clipboard. Paste it wherever you need it.
04
ankylo list
See all stored entries and their timestamps
