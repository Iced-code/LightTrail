How it works

Up and running in minutes

One script tag on the client. A small Node server on your machine. That's it.

01 —

Clone & configure

Clone the repo and add your Postgres connection string to .env.

02 —

Start the server

Run npm start. Express starts on port 3000 and creates the database table automatically.

03 —

Inject the script

Add one <script> tag to any HTML page. A LightTrail button appears instantly.

04 —

Highlight & comment

Select any text, type a comment, and save. All users on the page see it appear in real time.

Features

Everything you need

⚡

Real-time updates

Comments appear instantly for all users via WebSockets — no polling, no refresh.

🎯

DOM-anchored

Each comment is tied to a specific element. They survive page reloads and stay in the right place.

🔒

Self-hosted

Your data stays on your server. No third-party services, no accounts, no subscriptions.

👥

Multi-user aware

Your comments appear in red, others in yellow. Anonymous IDs are assigned automatically.

🖱️

Draggable dialogs

Comment boxes are freely draggable and positions are stored as percentages for any screen size.

📦

Zero client deps

One script tag. No frameworks, no build step on the site you're annotating.

Installation

Get started

Clone, configure, and run — or use Docker to skip the Postgres setup entirely.

# clone & install
git clone https://github.com/Iced-Code/LightTrail.git
cd LightTrail && npm install

# configure
cp .env.example .env
# edit DATABASE_URL in .env

# run
npm start
<!-- add before </body> -->
<script src="http://localhost:3000/lighttrail.js"></script>

<!-- remove this before publishing your site -->
